Seven awesome sizes and basic typographic styles

There are seven sizes: xxs (.5), xs (.75), s (.875), m (1), l (1.5), xl (2), xxl (4). Base size is 16 pixels; everything else builds from the base. Five font weights (thinner, thin, regular, bold, bolder).

The basic tags have their own base settings, but can also be used with classes to modify visual size and line-height (font weight is inherited from the defined tag settings to be consistent across all pages/documents).
